# Documentação do Projeto

Este diretório contém a documentação criada automaticamente que descreve o estado atual do projeto, como executar a aplicação, alterações principais e a definição dos campos do modelo `Anamnese`.

Arquivos incluídos:

- `PROJECT_DOCUMENTATION.md` — Visão geral do projeto, estrutura de apps e o que foi implementado.
- `SETUP_AND_RUN.md` — Como configurar o ambiente e executar o projeto localmente.
- `ANAMNESE_FIELDS.md` — Lista detalhada dos campos do modelo `Anamnese` com descrições.
- `CHANGELOG.md` — Histórico cronológico das alterações feitas durante este trabalho.

Como usar

- Leia `PROJECT_DOCUMENTATION.md` para entender a arquitetura e arquivos-chave.
- Siga `SETUP_AND_RUN.md` para iniciar o servidor localmente.
- Consulte `ANAMNESE_FIELDS.md` para ver os campos do formulário de anamnese e suas intenções.
- Veja `CHANGELOG.md` para o histórico de mudanças.
