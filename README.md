# Projeto Django (exportado)

Este repositório contém o projeto Django presente no workspace. Contém os apps `cadastro`, `consulta`, `dashboard` e a pasta `projeto` com as configurações.

Instruções rápidas:

1. Crie (ou ative) um ambiente virtual Python e instale dependências.

   ```powershell
   python -m venv .venv
   .\.venv\Scripts\Activate.ps1
   pip install -r requirements.txt   # se existir
   ```

2. Se for necessário, crie um repositório remoto no GitHub e me forneça a URL para eu configurar o remote e dar push, ou rode:

   ```powershell
   git remote add origin https://github.com/SEU_USUARIO/SEU_REPO.git
   git branch -M main
   git push -u origin main
   ```

Observações:
- Para eu empurrar (push) para o GitHub preciso que você me forneça a URL do repositório remoto ou que autorize a criação via GitHub CLI (`gh auth login` pode ser necessário).
